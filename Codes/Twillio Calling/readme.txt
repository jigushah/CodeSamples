in twillio,First we have to get one twillio number ,Some configuration is needed for calling and sms on twillio console,
After that twillio basiclly runs on SIP protocol It make a tunnel between two end point.
Console will provide use some keys for authentication,
Twillio make a token with the combination of our api key and the phone number,
With that generated token we can initiate call request,
Twillio SDK provide listeners for connected,disconnected,Hangup events.
